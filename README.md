# PowerModule Base on MP2236

MAX Input Voltage: 18V   
MAX Vout :16V(theoratically)   
Max Iout : 4A(with heatsink 5A)  
Suggested Vin 16V to 12V
Suggested Vout 5V to 3.3V
 
for high Vout and high Iout,heatsink is recommended.
Verified design,but haven't measure the ripple voltage yet.
